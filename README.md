## Installation

To start the project, clone the repository from GitHub and install the dependencies.

```
$ git clone https://github.com/nipek/frensei-mvp.git ./yourProjectName 
$ cd yourProjectName
$ npm install
```


## Start the app

```
$ expo start
```
